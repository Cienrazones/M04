Git

Basico

  1. git status — Verifica qué archivos han cambiado desde el último commit
  2. git add . — Añade todos los cambios al seguimiento (o selecciona archivos específicos)
  3. git commit -m "el cambio" — Mejor en ingles
  4. git push — Sube los cambios al repositorio remoto en GitHub

¿Cómo conectar mi proyecto local a un repositorio vacío en GitHub?

  1. git init — Inicializa Git en tu carpeta local
  2. git add . — Añade todos los archivos al seguimiento
  3. git commit -m "Initial commit" — Crea el primer commit
  4. git remote add origin https://github.com/tu-usuario/nombre-del-repo.git — Enlaza con el repositorio remoto
  5. git push -u origin main — Sube el proyecto a GitHub
