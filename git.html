Comandos útiles de Git

Flujo básico

  • git status → git add . → git commit -m "mensaje" → git push
  • git pull — traer cambios
  • git clone URL — copiar repo

Inicio rápido

  • git init → git add . → git commit -m "init"
  • git remote add origin URL
  • git push -u origin main

Ramas

  • git branch — ver
  • git checkout rama — cambiar
  • git merge rama — unir
  • git branch -d rama — borrar

Historial y cambios

  • git log --oneline
  • git diff
  • git reset --hard id
  • git revert id

.gitignore

  • *.log
  • node_modules/
  • assets/temp/

Notas rápidas

  • git commit -am "mensaje" — directo
  • git stash — guardar sin subir
  • git push origin rama — subir rama

Goku dice:

git status → add → commit → push

“No subas cajas vacías. Siempre revisa antes de empujar.”
